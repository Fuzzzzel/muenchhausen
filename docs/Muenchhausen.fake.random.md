### element() 

Returns a random element from Array or Object.  
JavaScript example: random.element({of:[1,2,3,4,5]})  
Template example:
```bash
$ mh "$(random.element of:[mh-1,mh-2,mh-3]) $(random.element) $(random.element of:{foo:5,unicorn:true,rainbow:blue})"
mh-1 ? true
```
